# Monitor 远程占用监控补齐计划

## 当前目标

把 Monitor 从“本机计时和管理工具”调整为“远程控制占用监控工具”。

核心目标是：检测当前机器是否正在被向日葵、ToDesk、AnyDesk 等远程控制软件使用，记录远程使用开始、结束、操作人，并把当前占用状态发布到网页，供其他远程使用者连接前查看，避免多人同时控制同一台机器。

## 当前状态

- `PRODUCT.md` 已修正为新的产品定位。
- 本计划只用于后续开发指导，当前不执行功能代码修改。

| 6. 本机端 UI 重构 | 待执行 | 首页改为远程占用监控台，计时、预约、统计降级为辅助 |
| 7. 历史记录和统计重构 | 待执行 | 以远程占用会话为核心重构历史和统计 |
| 8. 设置页补齐 | 待执行 | 增加远程工具检测、发布地址、刷新频率、访问密码等设置 |

## 不做的事

- 不把手动计时继续当成主功能。
- 不用含糊的“安全”代替远程占用状态。
- 不在无法识别操作人的情况下假装记录完整。
- 不用硬编码臆测进程名直接上线，必须经过本机真实验证或配置化。
